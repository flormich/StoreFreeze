## 💻 Installation
Clone this reposoitory

```
git clone  https://github.com/flormich/StoreFreeze

```

```json
{
    "index": {
        "method" : "get",
        "path" : "/project/public/",
        "controller" : "AppController",
        "action" : "show"  
    }
}
```

Change the database in `config/database.json` like in exemple :
```json
{
    "username" :  "root",
    "database" : "database_name",
    "host" : "localhost",
    "password" : "password"
}
```
